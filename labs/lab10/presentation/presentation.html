Цели и задачи работы

Цель лабораторной работы

Получить навыки работы с утилитами управления модулями ядра операционной системы Linux.

Задачи лабораторной работы

  1. Ознакомиться с командами для просмотра устройств и модулей ядра
  2. Научиться загружать и выгружать модули ядра
  3. Изучить параметры модулей с помощью modinfo
  4. Освоить обновление ядра системы
  5. Закрепить знания об администрировании и сопровождении ОС Linux
